## A React Native ripple component.

![](https://github.com/rgovind92/react-native-ripple/blob/master/preview.gif)

## Installation
```shell
npm install --save react-native-ripple
```

## Usage
```jsx
import Ripple from 'react-native-ripple';

...
  render (
    <Ripple>
      <CustomElement />
    </Ripple>
  )
```
